# Infinite Cycle Carousel Example

This example demonstrates the usage of the `infinite_cycle_carousel` package with various configurations and 9 different carousel styles.

## Features Demonstrated

1. **Basic Carousel with Auto Play** - Simple carousel that automatically cycles through items
2. **Carousel with Indicators** - Shows dot indicators below the carousel
3. **Carousel with Enlarge Effect** - Center item is enlarged for emphasis
4. **Controlled Carousel** - Manual navigation with previous/next buttons
5. **Vertical Carousel** - Scrolls vertically with line indicators
6. **3D Carousel** - Cylindrical 3D carousel with perspective effects and glass effects
7. **ViewPager Style** - Android ViewPager-style with overlapping cards
8. **Circular Carousel** - Unique circular positioning with depth and scale effects
9. **Stacked Carousel** - Stacked cards with glimpse of side items

## Running the Example

From the example directory, run:

```bash
flutter run
```

Or from the root package directory:

```bash
cd example
flutter run
```

## Code Highlights

The example showcases:
- 9 different carousel styles and variants
- Multiple indicator styles (dot, line, scale)
- Programmatic control via controller
- Custom styling and animations
- Both horizontal and vertical layouts
- Auto play with pause on touch
- Page change callbacks
- 3D transformations and perspective effects
- Circular positioning with radius-based layouts
- ViewPager-style overlapping cards
- Responsive and adaptive layouts
